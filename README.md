# 奶龙素材馆

这是《Web前端开发》期末大作业项目源码，主题为“奶龙素材馆”。

## 运行方式

直接用浏览器打开 `index.html` 即可运行。项目是纯静态页面，不需要安装依赖。

## 页面结构

- `index.html`：首页，包含页眉、导航、banner、真实素材推荐、副栏、页脚、视频、Canvas、浮动广告和拖放收藏。
- `login.html`：登录页，包含用户名和密码验证，并使用 `localStorage` 保存用户名。
- `contact.html`：人格测试页，填写占位测试题后随机生成“你是这个奶龙”的照片结果。
- `category.html`：画廊分类页，按 `images/photos/奶`、`images/photos/奶果`、`images/photos/奶气`、`images/gifs`、`images/奶龙字母` 筛选素材，并包含分页导航和拖放收藏。
- `detail.html`：收藏展示页，根据用户收藏内容展示画廊。
- `word.html`：奶龙组词页，输入 1-15 个英文字母，用奶龙字母素材拼成 Canvas 图片并下载。

## 文件说明

- `css/style.css`：所有页面的统一样式。
- `js/common.js`：用户状态、素材配置、素材列表加载等公共能力。
- `js/favorites.js`：拖放收藏、收藏列表渲染等跨页能力。
- `js/*.js`：每个页面绑定自己的脚本，例如 `index.js`、`category.js`、`contact.js`。
- `js/main.js`：兼容旧缓存 HTML 的兜底加载器，当前页面不再直接依赖它。
- `images/`：本地主题图片资源，其中 `photos/奶`、`photos/奶果`、`photos/奶气`、`gifs`、`奶龙字母` 会导入分类页。
- `report.md`：期末报告初稿，可按自己的姓名、学号和截图继续补充。

## 评分点对应

- 网页完整性：登录、测试、首页、分类、收藏等页面。
- 主页完整性：首页包含 logo、导航、banner、正文、副栏和页脚。
- 元素多样性：文字、列表、图片、视频、Canvas、表单和按钮。
- 数据验证：登录页和人格测试页均有 JavaScript 验证。
- 用户状态：首页、分类页、收藏页显示当前用户名。
- 图片切换：首页 banner 自动和手动切换。
- 图形绘制：首页使用 Canvas 动态绘制素材占比统计图。
- 浮动：首页右下角浮动提醒。
- 拖放：首页和分类页可拖动卡片到收藏篮。
- 分页导航：分类页支持分页显示，当前每页展示 12 个素材。
- 特色功能：奶龙组词页支持英文输入验证、Canvas 拼图和图片下载。
